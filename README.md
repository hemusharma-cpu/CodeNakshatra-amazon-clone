# 🛒 Amazon Clone (React.js)

A full-stack Amazon clone built using React.js. This project replicates core e-commerce functionalities like product browsing, cart management, authentication, and checkout.

---

## 🚀 Features

* 🔐 User Authentication (Login / Signup)
* 🛍️ Product Listing Page
* 🛒 Add to Cart / Remove from Cart
* 💳 Checkout System (Stripe / Dummy Payment)
* 📦 Order History
* 🔎 Search Functionality
* 📱 Responsive Design (Mobile + Desktop)

---

## 🧑‍💻 Tech Stack

**Frontend:**

* React.js
* React Router
* Context API / Redux
* Tailwind CSS / CSS

**Backend (optional):**

* Firebase / Node.js / Express

**Database:**

* Firebase Firestore / MongoDB

**Payment Integration:**

* Stripe API

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/your-username/amazon-clone.git
cd amazon-clone
```

Install dependencies:

```bash
npm install
```

Run the app:

```bash
npm start
```

---

## 🔑 Environment Variables

Create a `.env` file in the root directory and add:

```
REACT_APP_FIREBASE_API_KEY=your_key
REACT_APP_STRIPE_KEY=your_key
